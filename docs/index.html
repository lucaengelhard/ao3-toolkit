ao3-toolkit

A Toolkit for interfacing with the Archive of Our Own

Coded in TypeScript with nodejs

Features

  • Logging in to ao3
  • Fetching single works
    • Fetching work content
    • Fetching work stats
  • Fetching user history
  • Fetching user bookmarks
  • Sorting WorkLists
    • by words
    • by kudos
    • by hits
    • by bookmarks
    • by comments
    • by chaptercount
    • by times visited
    • by last time visited

Roadmap

  • Fetching work comments
  • Fetching user stats
    • words read
    • fics read
    • favourite tags
    • favourite fandoms
